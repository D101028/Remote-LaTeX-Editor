## Getting Started

1. Build the Python environment.

2. Create the virtual environment.
```shell
$ python3 -m venv .venv
```

3. Activate the virtual environment.
```shell
# Windows
$ .venv/Scripts/activate

# Unix / MacOS
$ source .venv/bin/activate
```

4. Start the app.
```shell
$ python3 main.py
```
